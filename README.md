# EduScore - Öğrenci Yönetim Sistemi (v2.5)

C programlama dili kullanılarak geliştirilmiş; dinamik bellek yönetimi, dosya işlemleri ve gelişmiş arama algoritmaları içeren kapsamlı bir öğrenci takip otomasyonu.


## Özellikler

Bu proje v2.5 sürümüyle birlikte **Clean Code** prensiplerine uygun hale getirilmiş ve güçlendirilmiştir:

- **Dinamik Bellek Yönetimi:** `malloc` ve `realloc` ile RAM verimli kullanılır.
- **Kalıcı Veri Tabanı:** Veriler `Sinif.txt` dosyasında saklanır, program kapansa bile silinmez.
- **Kesin Arama (ID):** Okul numarası ile nokta atışı öğrenci bulma özelliği.
- **Detaylı Sıralama:** Not ortalamasına veya numaraya göre başarı sıralaması.
- **Veri Bütünlüğü:** Struct yapıları sayesinde sıralama sırasında veri kayması yaşanmaz.

  
## Kullanılan Teknolojiler


- **Dil:** C Programming Language
- **Veri Yapıları:** Structs, Pointers, Arrays
- **Algoritmalar:** Bubble Sort, Linear Search
- **Dosya İşlemleri:** File I/O (Okuma/Yazma)
